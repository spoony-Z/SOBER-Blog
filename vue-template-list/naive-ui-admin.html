# 简介

Naive Ui Adminopen in new window 完全免费，且可商用，基于 Vue3.0open in new window、Viteopen in new window、 Naive UIopen in new window、TypeScriptopen in new window 的中后台解决方案，它使用了最新的前端技术栈，并提炼了典型的业务模型，页面，包括二次封装组件、动态菜单、权限校验、粒子化权限控制等功能，它可以帮助你快速搭建企业级中后台项目， 相信不管是从新技术使用还是其他方面，都能帮助到你。

# 特性

  • 二次封装实用高扩展性组件
  • 响应式、多主题，多配置，快速集成，开箱即用
  • 最新技术栈，使用 Vue3、Typescript、Pinia、Vite 等前端前沿技术
  • 强大的鉴权系统，对路由、菜单、功能点等支持三种鉴权模式，满足不同的业务鉴权需求
  • 持续更新，实用性页面模板功能和交互，随意搭配组合，让构建页面变得简单化

# 使用

  • 获取项目代码
git clone https://github.com/jekip/naive-ui-admin.git
  • 安装依赖
cd naive-ui-admin

yarn install
  • 运行
yarn dev
  • 打包
yarn build

# Git 贡献提交规范

  • 参考 vueopen in new window 规范 (Angularopen in new window)

    • feat 增加新功能
    • fix 修复问题/BUG
    • style 代码风格相关无影响运行结果的
    • perf 优化/性能提升
    • refactor 重构
    • revert 撤销修改
    • test 测试相关
    • docs 文档/注释
    • chore 依赖更新/脚手架配置修改等
    • workflow 工作流改进
    • ci 持续集成
    • types 类型定义文件更改
    • wip 开发中
